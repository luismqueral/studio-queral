DataSynth is a client-side data sonification tool that transforms JSON, CSV, and GeoJSON data into audio. Built with vanilla JavaScript, Web Audio API, and D3.js â€” no frameworks, no build step.

## What it does

You load a dataset (earthquakes, exoplanets, your own CSV), and DataSynth intelligently maps data fields to audio parameters:

- **Magnitude** â†’ Pitch
- **Depth** â†’ Filter frequency  
- **Latitude** â†’ Stereo pan
- **Time** â†’ Rhythm

The result is an audio representation of your data that reveals patterns you might not see in a spreadsheet.

## Features

**Intelligent Mapping** â€” Auto-analyzes your data to find interesting patterns. Maps fields to audio parameters and suggests optimal scaling curves based on data distribution.

**Dual Sound Engines** â€” Choose between a synthesizer (10 waveform types including FM, noise, additive) or a sampler (upload your own audio files, pitch shift and slice them with data).

**Real-Time Effects** â€” Reverb, delay with analog pitch-shifting, filter (4 types), stereo panning, musical quantization.

**Live Visualization** â€” A patch cable interface showing dataâ†’audio mappings, real-time waveform display, and an interactive node graph.

## Tech stack

The entire thing is 4 ES6 modules (~2000 lines total):

- `data-processor.js` â€” Pure functions for data parsing
- `parameter-mapper.js` â€” Intelligent mapping algorithm  
- `audio-engine.js` â€” Web Audio API management
- `patch-viz.js` â€” D3.js visualization

No build step. No frameworks. Deploy to any static host.

## Why I built this

I've always been interested in the intersection of data and sound. There's something satisfying about hearing patterns that you can't quite see â€” the way earthquake swarms cluster, or how exoplanet properties distribute across the galaxy.

This started as an experiment to learn the Web Audio API and turned into a surprisingly capable tool for exploratory data analysis through sound.
